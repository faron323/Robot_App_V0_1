Dual Arm Automated Conveyor System

--Raspberry Pi--

-I/O
  Serial usb connection to Arduino's.
  Bluetooth or USB connection with controller.

- X,Y Localisation
  Using either NN image recognition or Feature recognition (edge detection,
  colour filtering) to find item on belt.

- Task Delegation
  Decides which arm will be tasked with retrieving item.

-Controller
  Switch control between arms on belt.
  Control teaching mode on arms.
